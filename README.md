
# ⚔️ Knight Game  

A simple 2D knight game built using Unity 🎮  
لعبة الفارس تم تطويرها باستخدام Unity.

---

## 🎮 Game Idea | فكرة اللعبة

A game where you control a knight and explore or fight enemies.  
لعبة تتحكم فيها بفارس وتستكشف أو تقاتل الأعداء.

---

## 🕹️ Controls | التحكم

- WASD → Move | الحركة  
- Space → Jump | قفز  
- Mouse / Attack Button → Attack | هجوم  

---

## 🎯 Objective | الهدف

Defeat enemies and survive as long as possible.  
هزم الأعداء والبقاء لأطول وقت ممكن.
